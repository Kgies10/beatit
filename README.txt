Setup your project folder:

   BeatIt/
          beatit  (git repository)
          build   (build folder)
          install (prefix folder)

Copy the do-configure.sh file  from the git repository in your build folder.

BeatIt/build $ cp ../beatit/do-configure.sh my-configure.sh

Edit the my-configure.sh file
From your build folder 
  $ sh my-configure.sh
